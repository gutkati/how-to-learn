# Учебный проект "How-to-learn"  

- Проект создан на базе обучающего курса [Яндекс.Практикум](https://practicum.yandex.ru/profile/web/) Веб-разработчик. 
- Данный проект - это страница с советами и дополнительными материалами, котрые помогут повысить эффективность обучения.  
- Цель проекта - ознакомиться с принципами HTML, CSS, научиться декомпозировать структуру сайта с помощью методологии БЭМ.
***

### Технологии
- пути к файлам и организация файлов по БЭМ
- HTML
- CSS
