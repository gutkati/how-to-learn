# __how-to-learn__
___

Проект создан на базе обучающего курса [Яндекс.Практикум](https://practicum.yandex.ru/profile/web/) Веб-разработчик
При создании данного проекта использовались такие свойства как:
- пути к файлам и организация файлов по БЭМ
- HTML: iframe, ul, a, img, h1-h6, p
- CSS: Flexbof, Grid, transform, translate, animation, @keyframes, position
- подключение кода сторонних разработчиков
